# GRC Portfolio — Flourish Chidinma Ikwuegbu

## About Me
I am an accounting professional actively transitioning 
into Governance, Risk and Compliance (GRC), with 
hands-on cybersecurity experience gained through a 
security monitoring internship and structured 
self-directed study. This portfolio documents my 
monitoring experience, GRC framework application, 
and ongoing professional development as I build 
toward a career in risk management and security 
governance.
